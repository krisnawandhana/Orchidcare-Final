INSTRUKSI INSTALASI - ORCHIDCARE-FINAL
======================================

1. PERSYARATAN SISTEM
----------------------
Pastikan Anda sudah menginstal:

- Node.js (disarankan versi LTS)
- npm atau yarn
- Expo CLI

Untuk menginstal Expo CLI:
> npm install -g expo-cli


2. MENGUNDUH PROYEK
--------------------
Dari file ZIP:
> Ekstrak Orchidcare-Final-fix.zip
> Masuk ke folder: Orchidcare-Final


3. INSTALASI DEPENDENSI
-------------------------
> npm install
atau
> yarn install


4. MENJALANKAN APLIKASI
-------------------------
> npx expo start / npm run start

Expo Developer Tools akan terbuka di browser.
Scan QR code dengan aplikasi Expo Go di ponsel Anda untuk menjalankan aplikasi.
